# devBlog.kr - 프로젝트 개요

> **생성일**: 2026년 1월 9일  
> **유형**: 한국 IT 기업 기술 블로그 큐레이션 플랫폼

## 📝 프로젝트 설명

**devBlog.kr**은 한국의 다양한 IT 기업들의 기술 블로그 게시글을 자동으로 수집하고, 키워드 기반 태그 분류를 통해 개발자들이 원하는 정보를 쉽게 찾을 수 있도록 돕는 웹 플랫폼입니다.

### 핵심 가치

- **자동화**: 매일 00:00(KST)에 32개 기업 블로그에서 새 글을 자동 수집
- **태그 정확도**: 사전정의된 태그를 키워드 매칭으로 선택 (AI 비용 없음)
- **편의성**: 검색, 필터링, 북마크, 최근 본 글 등 사용자 중심 기능

### 참고한 서비스

- https://www.techblogposts.com/ko

## 🎯 프로젝트 목표

### 1. 개발자를 위한 정보 허브

한국 주요 IT 기업들의 기술 블로그 게시글을 한곳에서 확인할 수 있는 중앙 허브 역할

### 2. 효율적인 정보 탐색

태그 기반 필터링, 다중 회사 선택, 검색을 통해 원하는 기술 분야의 정보를 빠르게 찾을 수 있음

### 3. 자동화된 콘텐츠 수집

매일 자동으로 새 글을 수집하여 사용자가 놓치는 정보가 없도록 보장

### 4. 사용자 개인화

- GitHub OAuth 로그인
- 북마크 기능으로 나만의 읽기 목록 관리
- 최근 본 글 자동 저장
- 활동 히트맵으로 시각화
