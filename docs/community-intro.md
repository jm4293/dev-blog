# devBlog.kr 소개 (커뮤니티 홍보용)

## 🎯 한 줄 소개

**한국 IT 기업들의 개발블로그, 기술블로그, 테크블로그를 한 곳에서 모아보는 큐레이션 플랫폼**

---

## 💡 만든 이유

개발 공부를 하면서 토스, 카카오, 네이버 등 여러 기업의 기술 블로그를 돌아다니며 좋은 글을 찾는 게 번거로웠습니다.

"이런 개발블로그 글들을 한 곳에서 볼 수 없을까?" 라는 생각으로 시작했고,
**32개 기업의 기술블로그와 개발블로그**를 자동으로 수집해서 보여주는 플랫폼을 만들었습니다.

---

## ✨ 핵심 기능

### 1️⃣ 자동 수집

- 하루 2번 자동으로 블로그 글 수집 (GitHub Actions, KST 15:00 / 21:00)
- RSS 피드 기반 실시간 업데이트
- 중복 감지 & 키워드 기반 자동 태그 분류

### 2️⃣ 강력한 검색 & 필터

- **제목 검색**: 원하는 주제 즉시 검색
- **태그 필터**: React, Backend, DevOps 등 다중 선택
- **회사 필터**: 관심 있는 기업만 골라 보기
- **URL 공유**: 필터링 결과를 URL로 공유 가능

### 3️⃣ 편리한 관리

- GitHub OAuth 로그인
- 즐겨찾기 기능 (날짜별 그리드뷰)
- 최근 본 글 (로컬 저장, 최대 50개)
- 다크/라이트 모드

### 4️⃣ Push 알림 (계획 중)

- 새 글 등록 시 브라우저 알림
- 기기별 알림 설정 (OS 단위 그룹화)
- 만료된 구독 자동 정리

## 🛠 기술 스택

| 분류          | 기술                                           |
| ------------- | ---------------------------------------------- |
| **Frontend**  | Next.js 14, React 18, TypeScript, Tailwind CSS |
| **상태 관리** | Jotai (최소 사용), TanStack Query (주요)       |
| **Backend**   | Supabase (PostgreSQL, GitHub OAuth)            |
| **배포**      | Vercel (Cron Jobs)                             |
| **모니터링**  | Sentry, Google Analytics                       |

**특징:**
✅ 전체 TypeScript  
✅ Server Components 최대 활용  
✅ SSR/SSG 하이브리드  
✅ 반응형 디자인  
✅ Lighthouse 90+ 점수

## 📝 FAQ

**Q. 무료인가요?**  
A. 네! 완전 무료입니다. 광고도 없어요.

**Q. 원본 글은 어떻게 보나요?**  
A. 각 카드의 "전체 읽기" 링크를 클릭하면 원본 블로그로 이동합니다.

**Q. 로그인 없이도 사용 가능한가요?**  
A. 네! 로그인 없이 검색/필터링 모두 가능합니다. 로그인은 즐겨찾기 기능을 위해서만 필요해요.

**Q. 모바일 앱은 없나요?**  
A. 아직 없지만, 모바일 웹이 완벽하게 최적화되어 있어요. 향후 React Native로 앱 개발 계획 중입니다.

**Q. 새 블로그를 추가하고 싶어요.**  
A. [devblog.kr/request](https://devblog.kr/request)에서 요청하시면 검토 후 추가해드립니다!

**Q. 오픈소스인가요?**  
A. 네! MIT 라이선스입니다. 코드는 [GitHub](https://github.com/jm4293/dev-blog)에서 확인하실 수 있어요.
