## 📝 Product Hunt 용 (영문)

---

**Tagline**: Korean Tech Blog Aggregator - Stay Updated!

**Description**:
devBlog.kr is a curated platform that automatically collects and organizes tech blog posts from 32+ leading Korean IT companies including Toss, Kakao, Naver, Line, and Coupang.

**Key Features:**
✅ Auto-sync with 32+ company RSS feeds
✅ Advanced search & multi-tag filtering
✅ Bookmark your favorite posts (GitHub OAuth)
✅ Dark/Light mode
✅ Mobile-optimized responsive design
✅ 100% Free & Open Source

**Tech Stack:**
Next.js 14, TypeScript, Supabase, Vercel, TanStack Query

**Why we built this:**
As a developer, I found it tedious to visit multiple company blogs to stay updated on the latest tech trends. devBlog.kr solves this by aggregating all Korean tech blog posts in one place with powerful filtering.

**What makes it different:**

- Focuses on Korean tech ecosystem
- Keyword-based auto-tagging
- Real-time updates via Vercel Cron
- Clean, minimal UI with great UX

📰 RSS Feed: https://devblog.kr/feed.xml
