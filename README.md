## 💡 devBlog.kr이 뭔가요?

**개발 공부하면서 좋은 블로그 글 찾기 힘드셨죠?**

토스, 카카오, 네이버... 각 회사 홈페이지 일일이 돌아다니며 확인하기 번거로우셨나요?

**devBlog.kr**은 한국 주요 IT 기업들의 기술 블로그를 **매일 자동으로 수집**해서  
**한 곳에서 편하게 볼 수 있도록** 만든 무료 서비스입니다!

> 💬 "개발 공부할 때 정말 유용해요!"  
> 💬 "여러 블로그 돌아다닐 필요가 없어졌어요"  
> 💬 "태그 필터가 너무 편해요!"

## ✨ 무엇을 할 수 있나요?

### 📚 **한 곳에서 모든 기술 블로그 읽기**

- 32개 기업의 최신 글을 한눈에
- 하루 2번 자동으로 새 글 수집 (KST 15:00, 21:00)
- 광고 없이 깔끔하게!

### 🔍 **원하는 주제 쉽게 찾기**

- **검색**: "React", "백엔드" 같은 키워드로 검색
- **태그 필터**: Frontend, Backend, DevOps 등 분야별로 골라보기
- **회사 필터**: 관심 있는 회사만 골라보기

### ⭐ **마음에 드는 글 저장하기**

- GitHub 계정으로 로그인
- 좋은 글은 즐겨찾기로 저장
- 나중에 다시 찾아보기 편해요!

### 🌓 **편한 환경에서 읽기**

- 다크모드 / 라이트모드 지원
- 모바일, 태블릿, PC 모두 최적화
- 어디서든 편하게!

## 📱 어떻게 사용하나요?

### 1️⃣ 사이트 접속

**[devblog.kr](https://devblog.kr)** 접속하시면 바로 이용 가능해요!

### 2️⃣ 원하는 글 찾기

- 상단 검색창에서 **키워드 검색**
- 태그를 클릭해서 **관심 분야 필터링**  
  (예: React, TypeScript, 백엔드, DevOps...)
- 회사 로고를 클릭해서 **특정 회사 글만 보기**

### 3️⃣ 글 읽기

- 카드를 클릭하면 원본 블로그로 이동
- 개발 인사이트와 최신 트렌드를 확인하세요!

### 4️⃣ 즐겨찾기 (선택)

- GitHub로 로그인 (무료, 30초 소요)
- 마음에 드는 글에 ⭐ 즐겨찾기
- 나중에 다시 보기 편해요!
